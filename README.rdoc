= Description
Adds Ruby's powers to the command line.
Supports many enumerator methods applied to STDIN. The current line is represented by the line variable.

To get help:
  rubyc help
== Example
  $ ls | rubyc map 'line.upcase'
  GEMFILE
  RAILS_VERSION
  README.RDOC
  RAKEFILE
  ACTIONMAILER
  ACTIONPACK
  ...
